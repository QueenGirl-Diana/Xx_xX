My First Canvas


ctx will be used as the reference of the canvas for drawing

moveTo() method moves the point to the specified point in the canvas, without creating a line. Same as forward function which we had learned in code.org

syntax of moveTo - ctx.moveTo(x coordinate, y coordinate)

Change the x coordinates by 100 one by one till 600
for eg - ctx.moveTo(100, 0); then ctx.moveTo(200, 0); and so one

Change the y coordinates by 100 one by one till 500
for eg - ctx.moveTo(0, 100); then ctx.moveTo(0, 200); and so one

lineTo() method adds a new point and creates a line TO that point FROM the last specified point(which we have discussed in moveTo()) in the canvas

syntax of lineTo - lineTo(x coordinate, y coordinate)

Change the x and y coordinates randomly to show the changes

Note: The canvas tag is not supported in Internet Explorer 8 and earlier versions.
